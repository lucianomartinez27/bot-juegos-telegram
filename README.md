# BOT PARA TELEGRAM CON JUEGOS CLÁSICOS

## PARA PROBARLO Ingresá a [Bot de Juegos Clásicos](https://t.me/juegos_clasicos_bot)

### LISTA ACTUALIZADA DE JUEGOS

+ Por ahora los juegos disponibles en el bot son:
    - Ahorcado.
    - Mastermind (muertos y heridos).
    - Buscaminas.
+ En un futuro:
    - Piedras, papel o tijeras.

## Configuración

1.  Crea un archivo `.env` en la raíz del proyecto basándote en el archivo `.env.example`.
2.  Obtén un token de Telegram mediante [BotFather](https://t.me/botfather).
3.  Configura tu token en el archivo `.env`:
    ```
    TOKEN_TELEGRAM=TU_TOKEN_AQUI
    ```
